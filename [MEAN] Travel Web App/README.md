###README.docx includes project details and screenshots.

You can download the project and follow the below instructions if desired.
You will have to install Node.js and NPM first.

Backend Server:
 *  cd <backend_directory> /travlr
 *  npm install
 *  node . (or npm start)
 *  HTML site: http://localhost:3000
 *  JS site: http://localhost:3000/travel
 *  API Explorer (LoopBack): http://localhost:3000/explorer
 
Frontend Server:
 *  cd <frontend_directory> (app_admin)
 *  npm install
 *  ng serve
 *  http://localhost:4200
